# Analysis notebooks

The numbered notebooks define the intended execution and audit order. All notebooks are distributed without stored cell outputs; the corresponding precomputed summary outputs are in `../results/`, while complete run bundles are supplied as release assets.

`01_primary_hgctm_s_model.ipynb` is a revised public copy of the primary workflow. The superseded continent holdout, supervised classification, and classifier-based negative controls have been removed. The final geographic workflow is Notebook 08.

Notebook 09 and its completed manual reference table remain required before the final release.
