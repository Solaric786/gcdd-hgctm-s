# Results

This directory contains reviewable summary tables, figures, README files, environment records, and run manifests. Run-level `.npz`/`.npy` arrays and loss traces are intentionally stored in the compressed GitHub/Zenodo release assets instead of normal Git history.

The result folders map directly to Notebooks 00 and 02–08. The primary-model result bundle from Notebook 01 should be attached to the final release when regenerated. A manual-validation result folder must be added with Notebook 09.
